# Privacy Policy for Haron

**Last updated: May 2026**

Haron ("we", "us", or "our") is a file manager application designed for the Android platform. We are committed to protecting your privacy.

## 1. Information Collection and Use

Haron is designed with privacy as a core principle. 
- **Personal Data:** We do not collect, store, or transmit any personally identifiable information (PII).
- **Files and Credentials:** All your files, application settings, and network credentials (such as FTP/SMB passwords) are stored locally on your device.
- **Encryption:** Sensitive information like network credentials and files in the "Secure Folder" are encrypted using Android Keystore system and are inaccessible to us or any third party.

## 2. Network Features

Haron includes features that allow you to connect to remote servers (FTP, SMB, WebDAV, SSH) and cloud storage providers (Google Drive, Dropbox, Yandex Disk).
- **Direct Connection:** When you use these features, Haron connects directly to the servers you configure.
- **Third-Party Services:** For cloud storage, Haron uses official APIs and secure OAuth2 authentication. Your cloud data is only accessed to perform the actions you request (listing, downloading, uploading).

## 3. Analytics and Telemetry

Haron does **not** use any analytics, crash reporting, or telemetry services that send data to our servers or any third-party. 

## 4. Permissions

Haron requests several permissions to function as a file manager:
- **All Files Access:** Required to manage files on your device.
- **Network/Wi-Fi:** Required for network transfers and connecting to remote servers.
- **Camera:** Used solely for scanning QR codes for file transfer.
- **Biometrics:** Used for app locking and secure folder access if enabled by you.

None of these permissions are used to collect or harvest your data.

## 5. Third-Party Libraries

Haron uses various open-source libraries. Each library is used strictly for its intended purpose (e.g., VLC for media playback, JSch for SSH). We do not share your data with these library providers.

## 6. Children's Privacy

Our application does not collect any data, and therefore does not knowingly collect any data from children.

## 7. Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by updating the "Last updated" date of this Privacy Policy.

## 8. Contact Us

If you have any questions about this Privacy Policy, you can contact us via GitHub: https://github.com/Dolgoibimus/Haron

# Политика конфиденциальности Haron

**Дата последнего обновления: Май 2026**

Haron («мы», «нас» или «наш») — это приложение-файловый менеджер для платформы Android. Мы стремимся защищать вашу конфиденциальность.

## 1. Сбор и использование информации

Haron разработан с учетом принципов конфиденциальности.
- **Персональные данные:** Мы не собираем, не храним и не передаем никакую личную информацию (PII).
- **Файлы и учетные данные:** Все ваши файлы, настройки приложения и сетевые учетные данные (например, пароли FTP/SMB) хранятся локально на вашем устройстве.
- **Шифрование:** Конфиденциальная информация, такая как сетевые учетные данные и файлы в «Защищенной папке», шифруется с использованием системы Android Keystore и недоступна нам или любой третьей стороне.

## 2. Сетевые функции

Haron включает функции, позволяющие подключаться к удаленным серверам (FTP, SMB, WebDAV, SSH) и поставщикам облачных хранилищ (Google Drive, Dropbox, Яндекс Диск).
- **Прямое соединение:** При использовании этих функций Haron подключается напрямую к настроенным вами серверам.
- **Сторонние сервисы:** Для облачных хранилищ Haron использует официальные API и безопасную аутентификацию OAuth2. Доступ к вашим облачным данным осуществляется только для выполнения запрошенных вами действий (просмотр списка, загрузка, выгрузка).

## 3. Аналитика и телеметрия

Haron **не** использует никаких сервисов аналитики, отчетов об ошибках или телеметрии, которые отправляют данные на наши серверы или какой-либо третьей стороне.

## 4. Разрешения

Haron запрашивает несколько разрешений для работы в качестве файлового менеджера:
- **Доступ ко всем файлам:** Требуется для управления файлами на вашем устройстве.
- **Сеть/Wi-Fi:** Требуется для сетевых передач и подключения к удаленным серверам.
- **Камера:** Используется исключительно для сканирования QR-кодов для передачи файлов.
- **Биометрия:** Используется для блокировки приложения и доступа к защищенным папкам, если вы включили эти функции.

Ни одно из этих разрешений не используется для сбора или кражи ваших данных.

## 5. Сторонние библиотеки

Haron использует различные библиотеки с открытым исходным кодом. Каждая библиотека используется строго по назначению (например, VLC для воспроизведения мультимедиа, JSch для SSH). Мы не передаем ваши данные поставщикам этих библиотек.

## 6. Конфиденциальность детей

Наше приложение не собирает никаких данных и, следовательно, намеренно не собирает никаких данных о детях.

## 7. Изменения в настоящей Политике конфиденциальности

Мы можем время от времени обновлять нашу Политику конфиденциальности. Мы сообщим вам о любых изменениях, обновив дату «Последнего обновления» в настоящей Политике конфиденциальности.

## 8. Свяжитесь с нами

Если у вас есть вопросы по поводу этой Политики конфиденциальности, вы можете связаться с нами через GitHub: https://github.com/Dolgoibimus/Haron
