# v0.91 — Power saving, swipe-back, audio fixes
- Power save mode: disables animations, reduces background activity
- Battery threshold: auto-disable animations below set level (default 15%)
- Swipe back from left edge on all settings screens, terminal, library, storage, duplicates, tags
- Audio player: no autoplay, always starts from beginning, correct track on file tap
- Progress bar positioned above custom navbar
- Tag screens redesigned: separate create/edit screen, back navigation
- Code audit: logging gaps filled, KDoc comments, architecture docs

# v0.90 — Animated themes
- 4 overlay animations: Matrix Rain, Snowfall, Starfield, Dust Particles
- Individual settings per animation: speed, density, opacity, size
- Matrix Rain: 5 charsets, custom RGB color, glow effect
- "Only while charging" option for each animation
- Auto-pause when app in background or screen off
