# Навигация и интерфейс
- Двухпанельная вертикальная раскладка (верх + низ)
- Перетаскиваемый разделитель панелей с 3 зонами тапа (закладки / 50-50 / инструменты)
- Навигация по папкам с кнопкой «Назад»
- Кликабельные хлебные крошки для быстрого перехода
- История навигации по панели (← → как в браузере)
- Открыть текущую папку в другой панели
- Активная панель подсвечена рамкой
- Позиция прокрутки сохраняется для каждой панели

# Файлы и папки
- Список файлов с иконками по типу
- Миниатюры предпросмотра: изображения, видео, текст, PDF, документы, APK, FB2
- Масштабируемая сетка: 1–6 колонок щипком
- Тап-зоны в списке: тап → открыть, долгий тап слева → выделить, долгий тап справа → быстрая отправка, тап по иконке → предпросмотр
- Тап-зоны в сетке: тап по иконке → предпросмотр, долгий тап → выделить, тап по имени → открыть, долгий тап по имени → быстрая отправка
- Бейдж расширения файла на иконках
- Пустые папки визуально выделены
- Сортировка по имени, дате, размеру, расширению
- Показать/скрыть скрытые файлы
- Быстрый поиск в текущей папке
- Создание шаблонов: папка, TXT, MD, датированная папка

# Выделение и операции
- Долгий тап + свайп для выделения диапазона
- Выделить все / снять выделение (долгий тап → выделить по расширению)
- Счётчик файлов и общий размер выделения
- Копирование и перемещение между панелями
- Удаление в корзину
- Переименование прямо в списке (как в Windows)
- Пакетное переименование: шаблон, нумерация, поиск и замена
- Drag-and-Drop между панелями (одиночный и групповой)
- Выбор операции при перетаскивании: проведи файл через разделитель — левая сторона копирует, правая перемещает [NEW]
- Дублирование файлов: долгий тап по Копировать → диалог с количеством копий и 3 направлениями [NEW]
  - В подпапку здесь: создаёт папку с именем файла, копии внутри [NEW]
  - В другую панель (подпапка): то же, но в папке другой панели [NEW]
  - В другую панель (открытая папка): копии прямо в текущую папку другой панели [NEW]
  - Имена копий: файл(1).ext, файл(2).ext и т.д. [NEW]
  - Размер папки обновляется в реальном времени во время дублирования [NEW]
- Диалог конфликтов: заменить / переименовать / пропустить

# Корзина
- Удалённые файлы хранятся 30 дней
- Просмотр, восстановление, очистка корзины
- Автоочистка при превышении лимита
- Лимит размера: 0–5000 МБ
- Пофайловый прогресс удаления (показывает каждый удаляемый файл)
- Автовосстановление записей из файловой системы при повреждении метаданных

# Фоновые операции
- Foreground Service для больших операций (>10 файлов или >50 МБ)
- Уведомление с прогрессом и кнопкой отмены
- Полоса прогресса внизу экрана

# Боковое меню (Drawer)

- Оформление: системная / светлая / тёмная тема
- Настройки

# Временная полка

# Хранилище (SAF)

- Все файловые операции работают с content:// URI

# Архивы
- Просмотр ZIP, 7z, RAR (включая RAR5)
- Встроенный просмотр в панели как обычная папка
- Навигация по папкам внутри архива
- Выборочная распаковка
- Создание ZIP с опциональным паролем (AES-256) и разделением
- Диалог конфликтов при создании: заменить / переименовать / отмена
- Варианты распаковки: распаковать здесь или в подпапку по имени архива
- Умное определение: если в архиве одна корневая папка — распаковать как есть без диалога
- Быстрая распаковка из списка файлов: выдели архивы (ZIP/7z/RAR) — иконка Архив пульсирует и меняется на Извлечь [NEW]
- Тап по пульсирующей иконке → выбор куда: рядом с архивом / текущая панель / другая панель [NEW]
- Каждый архив распаковывается в подпапку с именем архива (создаётся автоматически) [NEW]
- Работает с одним или несколькими архивами одновременно [NEW]
- Архивация «1 в 1»: выдели несколько файлов → Архив → «1 в 1» — каждый файл в отдельный ZIP [NEW]

- Долгий тап по щиту → войти в защищённые файлы

- Виртуальная директория для всех защищённых файлов

# Глобальный поиск
