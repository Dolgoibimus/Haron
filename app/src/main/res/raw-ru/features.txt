# Навигация и интерфейс
- Двухпанельная вертикальная раскладка (верх + низ)
  > Портрет: панели расположены вертикально (верх + низ)
  > Ландшафт: панели расположены горизонтально (лево + право)
  > Тап в любом месте неактивной панели только активирует её — файл не откроется, следующий тап работает как обычно
- Перетаскиваемый разделитель панелей с 3 зонами тапа (закладки / 50-50 / инструменты)
  > Зажми разделитель и тяни вверх/вниз (или влево/вправо в ландшафте) для изменения размера
  > Левая зона (показывает кол-во файлов): тап → круговое меню с 6 слотами закладок
  > Центральная зона: тап → сброс панелей на равные 50/50
  > Правая зона (показывает кол-во файлов): тап → круговое меню инструментов (Корзина, Анализатор, Дубликаты, APK, Плеер, Ридер)
  > При перетаскивании файлов через разделитель: левая сторона = копировать, правая = переместить
- Навигация по папкам с кнопкой «Назад»
  > Кнопка ← в заголовке панели: переход к предыдущей папке из истории, или в родительскую если истории нет
  > Кнопка →: переход вперёд по истории
  > Обе кнопки становятся серыми когда навигация невозможна
  > Свайп от края можно настроить как «назад» в Настройки → Жесты
- Кликабельные хлебные крошки для быстрого перехода
  > Тап по любому сегменту пути — прыжок сразу на тот уровень
  > Текущая папка (последний сегмент) не кликабельна
  > Путь прокручивается горизонтально — автоматически показывает текущую папку
  > Справа показывается размер папки / общий объём хранилища
- История навигации по панели (← → как в браузере)
  > Каждая панель хранит свою независимую историю навигации
  > Тап ← — назад, → — вперёд, как в браузере
  > История работает и внутри архивов
- Открыть текущую папку в другой панели
  > Тап по меню (⋮) в заголовке панели
  > Выбрать «Открыть в другой панели»
  > Та же папка откроется в противоположной панели — удобно для сравнения или перемещения файлов
- Активная панель подсвечена рамкой
  > Активная: толстая рамка (2dp) основного цвета + слегка приподнятый фон
  > Неактивная: тонкая рамка (1dp) приглушённого цвета
  > Переключается автоматически по тапу, выделению или любому действию
- Позиция прокрутки сохраняется для каждой панели
  > Каждая панель запоминает позицию прокрутки при навигации и возврате
- Виджет на рабочий стол: список избранного, тап по папке → приложение открывается в ней
  > Долгий тап по рабочему столу → Добавить виджет → Haron Избранное
  > Показывает список избранных папок с путями
  > Тап по папке → Haron открывается сразу в этой папке
- Зарегистрирован как системный обработчик файлов: открытие из любого приложения, приём через «Поделиться»
  > Haron появляется в системном выборе «Открыть с помощью» для файлов, архивов, документов, медиа
  > Приём файлов через «Поделиться» из любого приложения
  > Автоопределение USB-накопителей при подключении

# Файлы и папки
- Список файлов с иконками по типу
- Миниатюры предпросмотра: изображения, видео, текст, PDF, документы, APK, FB2
- Масштабируемая сетка: 1–6 колонок щипком
  > Щипок в любом месте списка файлов для смены колонок (1–6)
  > 1 колонка = подробный список, 2+ = сетка с миниатюрами
- Тап-зоны в списке: тап → открыть, долгий тап слева → выделить, середина → перетащить, справа → отправка, иконка → просмотр
  > Тап по имени файла → открыть файл или войти в папку
  > Долгий тап слева → выделить/снять выделение
  > Долгий тап по середине → файл выделяется и сразу готов к перетаскиванию — тяни в другую панель не отпуская палец
  > Долгий тап справа → быстрая отправка (появляются кружки устройств)
  > Тап по иконке файла → быстрый предпросмотр
- Тап-зоны в сетке: тап по иконке → предпросмотр, долгий тап → выделить, тап по имени → открыть, долгий тап по имени → быстрая отправка
  > Тап по миниатюре → быстрый предпросмотр
  > Долгий тап по миниатюре → файл выделяется и готов к перетаскиванию; если не тянуть — остаётся выделенным
  > Тап по имени под миниатюрой → открыть файл
  > Долгий тап по имени → появляются кружки устройств для быстрой отправки; для папок — выделение
- Бейдж расширения файла на иконках
- Пустые папки визуально выделены
- Сортировка по имени, дате, размеру, расширению
  > Меню (⋮) → Сортировка → выбрать поле и направление
- Показать/скрыть скрытые файлы
  > Меню (⋮) → «Скрытые файлы», или голосовая команда «скрытые»
- Быстрый поиск в текущей папке
  > Тап по 🔍 в заголовке панели → ввести текст для фильтрации файлов по имени
  > Кнопка «Тт» справа — поиск по имени, кнопка «📄» — поиск по содержимому файлов
  > Долгий тап по 🔍 → глобальный поиск по всему устройству (отдельный экран)
- Создание шаблонов: папка, TXT, MD, датированная папка
  > Тап по кнопке «+» в заголовке панели → выбрать тип шаблона
  > Голос: «создать» / «новый»

# Выделение и операции
- Долгий тап + свайп для выделения диапазона
  > Долгий тап по файлу в левой зоне — файл выделяется. Не отпуская палец, веди вверх или вниз — все файлы по пути тоже выделяются
- Выделить все / снять выделение (долгий тап → выделить по расширению)
  > Иконка «Выделить все» на панели выделения
  > Долгий тап по «Выделить все» → выделить только файлы с таким же расширением
  > Голос: «выделить все» / «снять выделение»
- Счётчик файлов и общий размер выделения
  > Показывается в разделителе: «3 файла / 45 МБ»
- Копирование и перемещение между панелями
  > Выделить файлы → тап Копировать или Переместить → файлы идут в папку другой панели
  > Голос: «копировать» / «переместить»
- Удаление в корзину
  > Выделить → тап Удалить → файлы перемещаются в Корзину (восстановимы 30 дней)
  > Голос: «удалить»
- Переименование прямо в списке (как в Windows)
  > Долгий тап по имени файла → имя становится редактируемым, ввести новое, подтвердить
  > Голос: «переименуй [имя]»
- Пакетное переименование: шаблон, нумерация, поиск и замена
  > Выделить несколько файлов → меню (⋮) → Пакетное переименование
  > Три режима: шаблон с переменными, последовательная нумерация, поиск и замена
- Drag-and-Drop между панелями (одиночный и групповой)
  > Долгий тап по файлу (средняя зона в списке, миниатюра в сетке) — файл выделяется и «прилипает» к пальцу
  > Не отпуская, тяни в другую панель — файл перемещается туда
  > Если до этого уже были выделены файлы, перетаскивание одного захватывает всё выделение
- Выбор операции при перетаскивании через разделитель
  > При перетаскивании файла через разделитель — отпусти на левой стороне = копировать, на правой = переместить
  > Дублирование: долгий тап по Копировать → количество копий и направление
  > 3 направления: подпапка здесь, другая панель (подпапка), другая панель (открытая папка)
  > Имена копий: файл(1).ext, файл(2).ext и т.д.
  > Размер папки обновляется в реальном времени
- Дублирование файлов
  > Долгий тап по Копировать → диалог с количеством копий и 3 направлениями
  > В подпапку здесь: создаёт папку с именем файла, копии внутри
  > В другую панель (подпапка): то же, но в папке другой панели
  > В другую панель (открытая папка): копии прямо в текущую папку другой панели
  > Имена копий: файл(1).ext, файл(2).ext и т.д.
  > Размер папки обновляется в реальном времени во время дублирования
- Диалог конфликтов: заменить / переименовать / пропустить

# Корзина
- Удалённые файлы хранятся 30 дней
  > Удалённые файлы не исчезают сразу — хранятся 30 дней, после чего удаляются автоматически
- Просмотр, восстановление, очистка корзины
  > Как открыть: Боковое меню → Корзина, или Круговое меню инструментов (правая зона разделителя), или голосом «корзина»
  > Экран корзины: список удалённых файлов с датой удаления и исходным путём
  > Выделить файлы → тап «Восстановить» → файлы возвращаются в исходную папку
  > «Очистить всё» → безвозвратно удаляет все файлы из корзины
  > «Удалить навсегда» → удаляет только выделенные файлы безвозвратно
- Автоочистка при превышении лимита
  > Когда корзина превышает установленный лимит — самые старые файлы удаляются автоматически
- Лимит размера: 0–5000 МБ
  > Настроить: Настройки → Лимит корзины (ползунок от 0 до 5000 МБ)
  > 0 = без ограничений
- Пофайловый прогресс удаления
  > При удалении папки с множеством файлов — показывается имя каждого удаляемого файла и прогресс
- Автовосстановление записей при повреждении метаданных
  > Если метаданные корзины повреждены (краш, нехватка места) — файлы не теряются
  > Haron находит «сиротские» файлы в папке корзины и восстанавливает записи автоматически

# Фоновые операции
- Все файловые операции работают в Foreground Service
  > Копирование, перемещение, удаление, архивация, извлечение — всё выполняется в фоновом сервисе
  > Можно смахнуть Haron из списка приложений — операция продолжится до завершения
  > В шторке уведомлений: прогресс-бар + имя текущего файла + кнопка отмены
- Полоса прогресса внизу экрана
  > По умолчанию свёрнута в тонкую полоску — видна, но не мешает работать
  > Тап по полоске → разворачивается: имя файла, скорость, проценты, кнопка отмены
  > Автоматически сворачивается при начале новой операции
- Пофайловый прогресс для операций с папками
  > При копировании/перемещении/удалении папки — показывается каждый файл отдельно, а не просто «папка»
- Скорость передачи во всех прогресс-барах (кроме удаления)
  > Отображается в МБ/с или КБ/с в реальном времени
- Кнопка отмены немедленно останавливает операцию
  > Тап «Отмена» → операция прекращается, уже обработанные файлы сохраняются

# Боковое меню (Drawer)
- Хранилища: внутреннее + SD-карты
  > Как открыть: тап по иконке ☰ в заголовке верхней панели, или свайп от левого края (если настроен в жестах), или голосом «меню»
  > Список всех доступных хранилищ: внутреннее, SD-карты, USB-накопители
  > Тап по хранилищу → панель переходит в корень этого хранилища
- Избранное: избранные папки
  > Быстрый доступ к закреплённым папкам
  > Добавить в избранное: меню (⋮) в заголовке панели → иконка звезды
  > Тап по избранной папке → панель переходит туда
- Недавние: 5 последних посещённых папок
  > Автоматически запоминает последние 5 папок, в которых вы были
- Инструменты: корзина, анализатор, дубликаты, пустые папки, принудительное удаление, метки, передача, терминал
  > Полный список инструментов — тап открывает соответствующий экран
- Настройки
  > Настройки приложения: тема, шрифт, иконки, безопасность, жесты, Shizuku

# Временная полка
- Буфер для перемещения файлов между далёкими папками
  > 1. Выделить файлы → тап «Полка» на панели выделения — файлы сохраняются на полку
  > 2. Перейти в нужную папку (хоть в другое хранилище)
  > 3. Открыть полку: Боковое меню → Полка, или голосом «полка»
  > 4. Тап «Вставить» — файлы копируются, «Переместить» — перемещаются
  > Полка сохраняется между сессиями — можно закрыть приложение и вернуться позже

# Хранилища и накопители
- Поддержка SD-карт и внешних накопителей
  > SD-карта появляется в Боковом меню автоматически при установке
  > Все файловые операции (копирование, перемещение, удаление) работают с SD-картой так же как с внутренним хранилищем
- Все файловые операции работают с content:// URI
  > Техническая особенность Android: на SD-картах и внешних накопителях используется SAF (Storage Access Framework)
  > Haron работает с ним прозрачно — для вас разницы нет
- Доступ к Android/data и Android/obb через Shizuku (Android 11+)
  > Проблема: Android 11+ блокирует доступ к папкам Android/data и Android/obb даже с полным доступом к хранилищу
  > Решение: Shizuku — бесплатное приложение, которое обходит это ограничение
  > 1. Установить Shizuku из Play Store
  > 2. Запустить Shizuku через ADB или беспроводную отладку (инструкция внутри Shizuku)
  > 3. В Haron: Настройки → Shizuku → Выдать разрешение
  > После этого: копирование, перемещение, удаление, переименование в Android/data и Android/obb работают полностью
- Автоопределение USB-накопителей (OTG)
  > Подключил флешку через OTG-переходник → Haron обнаруживает автоматически
  > USB-накопитель появляется в Боковом меню
- Навигация по USB, безопасное извлечение
  > Работа с файлами на флешке как с обычной папкой
  > Безопасное извлечение: иконка «Извлечь» рядом с USB-накопителем в Боковом меню
- Индикатор свободного/общего пространства
  > Для каждого хранилища показывается сколько занято и сколько всего
- [NEW] Поддержка ext4 USB без root
  > Первый Android файловый менеджер с чтением/записью ext4 USB без root
  > Подключите Linux-флешку (ext4) → Харон монтирует автоматически через lwext4
  > Все файловые операции: копирование, перемещение, удаление, переименование, создание папок
  > Миниатюры изображений загружаются по мере прокрутки с кешем на диске
  > Быстрый просмотр фото в полном разрешении
  > Копирование файлов с телефона на ext4 и обратно
  > Поблочная запись для больших файлов — без проблем с памятью
  > Настройки → размер кеша превью ext4 (0–500 МБ)
  > Поддержка ext4, ext3, ext2
  > Работает на всех устройствах — Android не поддерживает ext4 USB из коробки
# Быстрый просмотр
- Тап по иконке файла → диалог предпросмотра
  > В режиме списка: тап по иконке слева от имени файла
  > В режиме сетки: тап по миниатюре
  > Открывается диалог поверх панелей — быстро посмотреть содержимое без перехода на отдельный экран
- Поддерживаемые форматы: изображения, видео, аудио, текст, PDF, архивы, APK, документы, FB2
  > Изображения: показывает фото с возможностью свайпа к соседним файлам в папке
  > Видео/аудио: воспроизведение прямо в диалоге
  > Текст: первые строки файла
  > PDF: первая страница
  > APK: иконка + версия + разрешения
  > Архивы: список файлов внутри
- Кнопки внизу диалога
  > «Галерея» — открыть изображение в полноэкранной галерее
  > «Ридер» — открыть документ/PDF в ридере
  > «Архив» — открыть архив как папку в панели
  > «Плеер» — открыть медиафайл в полноэкранном плеере
- Удаление изображений прямо из просмотра
  > Кнопка корзины (🗑) в диалоге превью → файл удаляется в корзину → автоматический переход к следующему фото
# Медиаплеер
- Встроенный движок VLC (LibVLC)
- Все популярные кодеки: MP3, FLAC, OGG, WAV, AVI, MKV, MP4 и др.
- Плейлист по папке с автоперемоткой
  > Открой любой медиафайл → все медиафайлы в той же папке становятся плейлистом
  > Треки переключаются автоматически, режим повтора: Все → Один → Выкл
- Seekbar + перемотка ±10 сек
  > Тап по левой зоне → -10с, правой → +10с, центр → пауза/воспроизведение
- Позиция сохраняется между сессиями
- Настройки плеера: отдельные табы для видео и аудио
  > Тап по шестерёнке (⚙) в заголовке плеера → полноэкранный экран настроек
  > Табы Видео и Аудио с независимыми настройками DND
- Режим Не беспокоить: блокировка уведомлений при воспроизведении с тонкой настройкой
  > Выбор что пропускать: звонки, сообщения, будильники, повторные звонки
  > Выбор кто может дозвониться: никто, избранные, все контакты, все
  > Блокировка визуальных эффектов: всплывающие, статус-бар, включение экрана
  > Беззвучный режим: отключение вибрации при воспроизведении
## Видео
- Полноэкранный режим, иммерсивный режим
- Аппаратное ускорение видео (MediaCodec) — плавное воспроизведение больших MKV (7ГБ+)
- Свайп-жесты: яркость (левая половина), громкость (правая), выход (от левого края)
  > Свайп вверх/вниз по левой половине экрана → регулировка яркости (0–100)
  > Свайп вверх/вниз по правой половине → регулировка громкости (0–100)
  > Свайп вправо от левого края → выход из плеера

# Поиск
- Глобальный поиск по всему устройству
  > Как открыть: долгий тап по 🔍 в заголовке панели, или голосом «поиск» / «найди»
  > Экран поиска: поле ввода сверху, под ним фильтры-чипы, ниже результаты
- Два режима поиска: по имени и по содержимому
  > Кнопка «Тт» справа от поля ввода — ищет по имени файла (работает всегда, без индексации)
  > Кнопка «📄» — ищет текст ВНУТРИ файлов (нужна индексация, см. ниже)
  > Переключение между режимами — один тап по кнопке
- Результаты поиска
  > Каждый результат: иконка по типу файла (цветная), имя, путь к папке, размер, дата
  > Тап по результату → открыть файл
  > Долгий тап по результату → перейти к файлу в проводнике (покажет папку где лежит файл)
  > Тап по иконке результата → быстрый предпросмотр
  > В режиме «📄»: под результатом — фрагмент текста с найденным словом (тап по стрелке раскрывает)
  > Результаты загружаются по 200 штук — скролль вниз для следующей порции
- Фильтры: категория, размер, дата
  > Три ряда чипов-фильтров, прокручиваются горизонтально
  > Категории: Все · Документы · Изображения · Аудио · Видео · Архивы
  > Размер: <1МБ · 1-10МБ · 10-100МБ · 100М-1Г · >1ГБ
  > Дата: Сегодня · Неделя · Месяц · Год · Старше
  > Тап по чипу включает фильтр, повторный тап — выключает. Фильтры комбинируются
- Поиск по содержимому: текст, DOCX, ODT, PDF
  > Переключи на «📄» → вводишь слово → находит его внутри документов
  > Работает только после индексации (кнопка 🔄 в шапке)
  > Без индексации ищет только по имени файла
- Поиск по метаданным аудио/видео
  > Находит по тегам: исполнитель, альбом, название трека
  > Работает после медиа-индексации (кнопка 🎵)
- ML Kit распознавание объектов на фото
  > Вводишь «собака» — находит фото где есть собака
  > Работает после визуальной индексации (кнопка 📷)
- OCR на фотографиях (распознавание текста)
  > Вводишь текст — находит фото где этот текст виден (вывески, скриншоты, документы)
  > Работает после визуальной индексации (кнопка 📷)
- Поиск в архивах (имена файлов внутри)
  > Находит файлы внутри ZIP/RAR/7z по имени, без распаковки
- Индексация: зачем и как
  > Без индексации работает только поиск по имени файла — быстро, но ограничено
  > Индексация «учит» поисковик содержимому файлов — после неё можно искать по тексту, тегам, объектам на фото
  > Три кнопки в шапке экрана поиска — каждая отвечает за свой тип:
  > 🔄 Базовая: читает текст документов, EXIF фото, содержимое архивов
  > 🎵 Медиа: читает теги аудио/видео (длительность, битрейт, кодек, исполнитель)
  > 📷 Визуальная: распознаёт объекты на фото (ML Kit) + текст на фото (OCR)
  > Тап по кнопке → запускается индексация с полосой прогресса и именем текущего файла
  > Первый раз: может занять несколько минут (зависит от количества файлов)
  > Статус внизу: «Проиндексировано: X файлов · Y назад» + размер индекса содержимого
  > Если некоторые файлы не удалось обработать — появится «Пропущено: N файлов» (тап раскрывает список)
- Фоновое обновление индекса
  > Индекс обновляется автоматически при изменении файлов и включении экрана (каждые 30 мин)
  > Не нужно запускать индексацию вручную каждый раз
- Локальный поиск (в панели)
  > Тап по 🔍 в заголовке панели → быстрая фильтрация файлов в текущей папке по имени
  > Кнопка «📄» — переключает на поиск внутри файлов (TXT, DOCX, ODT, PDF и др.)
  > Первый поиск по содержимому индексирует папку (пару секунд), последующие — мгновенные
  > Разница с глобальным: локальный ищет только в текущей папке, глобальный — по всему устройству

# Безопасность
- PIN (4–8 цифр) с автовводом
  > Настроить: Настройки → Безопасность → выбрать длину PIN (4–8 цифр)
  > Автоввод: PIN подтверждается автоматически при вводе нужного количества цифр
- Биометрия (отпечаток пальца)
- Комбинированный: биометрия + PIN как запасной
- Тайм-аут блокировки (30 мин)
- Секретный вопрос для сброса PIN
- Защита скриншотов при блокировке
- Защищённая папка: шифрование Tink Streaming AEAD (AES-256-GCM-HKDF, сегменты 1МБ)
  > Файлы зашифрованы ключом Android Keystore — без устройства прочитать невозможно
  > Шифрование работает на любых размерах файлов без проблем с памятью (потоковая обработка по 1МБ)
- Кнопка щита — всегда на одном месте в заголовке панели (справа)
  > Тап + файлы выделены (не защищены) → зашифровать и защитить выделенные файлы
  > Тап + в режиме защиты + нет выделения → выход из режима защиты
  > Долгий тап (обычный режим) → аутентификация (PIN/биометрия) → вход в защищённую папку
  > Долгий тап + в режиме защиты + файлы выделены → расшифровать и снять защиту
- Защищённые файлы полностью невидимы без разблокировки щита
- PIN или биометрия для входа в режим защиты
- Виртуальная директория со всеми защищёнными файлами после аутентификации
- Копирование расшифровывает, перемещение расшифровывает + снимает защиту

# Сеть и передача
- Передача между устройствами: Wi-Fi Direct + NSD
  > Как открыть: Боковое меню → Передача, или голосом «передача»
  > Устройства с Haron в одной сети обнаруживаются автоматически
  > Выделить файлы → тап по устройству в списке → передача начинается
  > Haron автоматически выбирает лучший протокол (Wi-Fi Direct для скорости, NSD для надёжности)
- QR-передача — два режима (тап по QR переключает)
  > Режим Wi-Fi (оба устройства в одной сети): QR со ссылкой для скачивания
  > Режим точки доступа (разные сети или нет Wi-Fi):
  > Таб «Haron»: один QR — второй Haron сканирует, сам подключается и скачивает
  > Таб «Другие»: два QR для любого устройства — первый для Wi-Fi, второй со ссылкой
  > Точка доступа временная, выключается автоматически при закрытии диалога
- Прогресс, возобновление, повтор при обрыве
  > Полоса прогресса с процентами и скоростью
  > При обрыве соединения — автоматический повтор
  > Передача работает в Foreground Service — продолжается даже если свернуть приложение
- Быстрая отправка
  > В списке файлов: долгий тап по правой зоне файла → вокруг пальца появляются кружки с иконками устройств
  > Не отпуская палец, тяни к нужному устройству → файл отправляется
  > Доверенные устройства принимают файлы автоматически без подтверждения
  > Переименование устройств: можно назначить псевдоним каждому устройству
- SMB-браузер
  > Боковое меню → Передача → SMB
  > Автообнаружение компьютеров и NAS в локальной сети
  > Просмотр расшаренных папок в двухпанельном режиме — как локальные файлы
  > Загрузка и выгрузка файлов с полосой прогресса
  > Сохранённые подключения с зашифрованными учётными данными
  > Никнеймы серверов: переименование сохранённых серверов для удобства
  > Гостевые подключения сохраняются автоматически с пометкой «(Гость)»
- Сетевой стриминг медиа
  > Тап на видео или аудио в FTP/SMB/WebDAV/SFTP → воспроизводится сразу без скачивания
  > Облака (Яндекс, Google Drive, Dropbox) — стриминг через HTTP-прокси
  > Полный плейлист: все медиа-файлы в папке добавляются в плейлист
  > FTP/SFTP: HTTP-прокси обходит баг VLC с кириллицей в путях
  > SMB: прямой стриминг через smb:// протокол
  > WebDAV: прямой стриминг через http/https
- FTP-клиент
  > Подключение к удалённым FTP-серверам
  > Просмотр, скачивание, загрузка файлов с прогрессом
  > Двухпанельный режим: FTP-файлы сверху, локальные снизу
  > Поддержка FTPS (TLS)
  > Сохранённые FTP-подключения (зашифрованные учётные данные)
- FTP-сервер
  > Раздавать файлы телефона на ПК
  > Анонимный или парольный доступ
  > Режим «только чтение»

# ТВ
- Трансляция на Chromecast, Miracast, DLNA
  > Как начать: тап по иконке Cast (📺) в плеере, или выделить файлы → Cast в панели выделения
  > ТВ-устройства обнаруживаются автоматически в вашей сети
  > Выбрать устройство → контент начинает воспроизводиться на ТВ
- Режимы трансляции
  > Медиа: видео и аудио файлы на ТВ
  > Слайдшоу: выделить фото → Cast → автоматическая смена фото
  > PDF-презентация: PDF-файл отображается постранично на ТВ
  > Зеркало экрана: экран телефона транслируется через браузер на ТВ
- Пульт управления
  > Во время трансляции: play/pause, перемотка, регулировка громкости
- Транскодирование
  > Если видео в формате, который ТВ не поддерживает (AVI, MKV, WMV, MOV, FLV, 3GP, TS) — Haron автоматически конвертирует в MP4
  > Прогресс транскодирования с процентами
  > Кеш: уже транскодированные файлы воспроизводятся мгновенно при повторном касте
- Автофоллбэк аудио
  > Если ТВ не поддерживает аудиокодек (AC3/DTS) — автоматическая конвертация звуковой дорожки
- Фоновое воспроизведение
  > Каст продолжается при выключенном экране телефона
  > Транскодирование тоже продолжается в фоне — не нужно держать экран включённым
  > Wi-Fi остаётся активным через WifiLock, CPU через WakeLock
- Bluetooth HID пульт
  > Режим тачпада: перемещение курсора на ТВ, тап, долгий тап, прокрутка
  > Виртуальная клавиатура: ввод текста на экране ТВ (латиница + кириллица)
  > Подключение к спаренным Bluetooth-устройствам

# Терминал
- Встроенный терминал
  > Одно окно, прямой ввод с клавиатуры
  > Тап по терминалу показывает клавиатуру
  > Шрифт DejaVu Sans Mono с полной поддержкой Unicode
- Persistent shell с настоящим PTY
  > Один bash-процесс живёт всё время — export, alias, cd сохраняются между командами
  > Настоящий PTY (pseudoterminal) через нативный JNI
  > Ctrl+C мгновенно прерывает команду (реальный SIGINT через PTY)
- Полноэкранные приложения: vi, nano, htop, Claude Code
  > Полная эмуляция VT100/xterm: альтернативный экранный буфер, scroll regions, позиционирование курсора
  > vi, nano, htop, less, Claude Code — всё работает корректно
  > Сохранение/восстановление курсора, origin mode, auto-wrap mode
- Цветной вывод (16/256/RGB)
  > Полная поддержка ANSI: цвета, курсор, очистка экрана, вставка/удаление строк
  > Bold, italic, underline, reverse video
- SSH-подключение
  > Кнопка SSH в заголовке → диалог подключения (хост, пользователь, порт, пароль)
  > Пароль шифруется AES-256, можно сохранить
  > Keepalive каждые 30 сек — сессия не отваливается при простое
  > Полноценный интерактивный shell — запуск Claude Code, vi, htop на удалённом сервере
  > Кнопка отключения в заголовке — возврат в локальный shell
- Панель быстрых клавиш
  > Esc, Tab, ^C, ^D, ^Z, ^L, Paste, ↑↓←→, Home, End
  > Символы: ~ / | > < & ; " ' . - _
  > Работает и в локальном shell, и в SSH
- История команд
  > Долгий тап ↑ — выезжающий список последних команд (до 200)
  > Тап по команде — вставляет в терминал
  > Кнопка удаления (✕) рядом с каждой командой
  > Команды захватываются автоматически из буфера терминала при нажатии Enter
  > Пароли исключаются из истории

  > Одно действие нельзя назначить одновременно на тап и долгий тап
- Кастомные иконки кнопок
  > Настройки навбара → иконка палитры → экран «Иконки кнопок»
  > Тап на действие → выбор картинки из галереи → редактор обрезки (квадрат/круг, зум/пан)
  > Кастомная иконка заменяет стандартную в навбаре
  > Кнопка сброса (✕) для возврата к стандартной иконке
- Навигация стрелками (Курсор ↑↓←→)
  > Курсор ↑/↓: тап = курсор вверх/вниз по файлам, долгий тап ↑ = переключить панель, долгий тап ↓ = включить/выключить Shift
  > Курсор ←/→: тап ← = назад по истории, тап → = вперёд, долгий ← = родительская папка, долгий → = зайти в папку под курсором
  > Курсор подсвечивает файл рамкой, файл выделяется (полная заливка как в Windows)
  > Авто-прокрутка: файл под курсором всегда виден
  > Первый тап начинает с видимого файла, а не с начала списка
- Shift-режим (мульти-выделение)
  > Долгий тап ↓ = включить/выключить Shift (кнопка подсвечивается)
  > Shift ВКЛ: все файлы от начальной точки до курсора выделяются диапазоном
  > Shift ВЫКЛ: выделение сохраняется, обычный курсор продолжает работу
  > Мульти-диапазон: включите Shift в новом месте чтобы выделить ещё группу файлов
  > Удобно для выбора нескольких групп файлов без касания экрана

# Настройки
- Тема: системная / светлая / тёмная (Material Design 3)
  > Как открыть: Боковое меню → Настройки, или голосом «настройки»
  > Три режима оформления: следовать системной теме, всегда светлая, всегда тёмная
- Ночной режим по расписанию
  > Автоматическое переключение на тёмную тему по времени
  > Настроить время начала и окончания
- Ползунки размера шрифта и иконок
  > Увеличить или уменьшить размер текста и иконок во всём приложении
  > Ползунок от 60% до 140% для каждого
- Переключатель вибрации
  > Включить/выключить тактильную обратную связь (вибрация при выделении, щипке и др.)
- Переключатель запроса PIN при запуске
  > Если включено — при каждом запуске Haron запрашивает PIN или отпечаток
- Лимит размера корзины
  > Ползунок: 0–5000 МБ (0 = без ограничений)
  > При превышении — самые старые файлы удаляются автоматически
- Статус и управление Shizuku
  > Показывает: установлен ли Shizuku, запущен ли, выдано ли разрешение
  > Кнопки: установить, запустить, выдать разрешение — всё в одном месте
- Настройки навбара
  > Настройка кастомного навбара: страницы, кнопки, действия, кастомные иконки
  > Подробнее в разделе «Кастомный навбар»
- Настройки жестов и голоса
  > Настройка 4 свайпов от краёв экрана + голосовые команды
- Анимированные темы (Настройки → Анимации)
  > 4 анимации-оверлея с живым превью в карточках
  > Анимации рендерятся поверх всех фонов — текст, кнопки и иконки остаются видимыми
  > Только одна анимация активна одновременно (остальные авто-выключаются)
  > [NEW] Работает везде кроме галереи, видео/аудио плеера и читалок документов
- Матричный дождь
  > Падающие символы (Катакана, Бинарный, Латиница, Кириллица, Микс)
  > Цвет: 8 пресетов + свой RGB, скорость, плотность, прозрачность

  > Крупные снежинки падают быстрее (реалистичная физика), глобальный ветер

  > Ползунки скорости, плотности, прозрачности, размера

  > Каждая снежинка уникальна (процедурная генерация по seed)

  > Нет автовоспроизведения при открытии файла — нажмите play когда готовы

- [NEW] Резервное копирование (Настройки → Резервное копирование)
  > Создание бэкапа всех данных: настройки, сетевые учётки, облачные аккаунты, библиотека книг, защищённая папка
  > Выбор секций: Настройки, Учётки, Книги, Защищённая папка
  > Опциональный пароль: галочка «Защитить паролем» для Настроек/Учёток/Книг
  > Защищённая папка обязательно с паролем (файлы расшифровываются из Keystore устройства в бэкап)
  > Учётки (FTP/SMB/SSH/WebDAV/Облака) расшифровываются из Keystore устройства и шифруются паролем
  > Бэкап сохраняется в Download/HaronBackup/ как .zip (без шифрования) или .hbk (с шифрованием)
  > Бэкапы переживают переустановку APK — хранятся в публичной папке Download
  > Восстановление из бэкапа: ввод пароля если зашифрован, все данные записываются обратно
  > После восстановления: отчёт что восстановлено и какие есть проблемы (файлы не найдены, токены протухли)
  > Теги/избранное/закладки, указывающие на несуществующие файлы, помечаются предупреждением
  > Защищённая папка: разворачиваемый список файлов с чекбоксами и размерами
  > Пофайловый выбор: можно выбрать какие именно файлы включить в бэкап
  > Прогресс в % отображается в кнопке «Создать бэкап» во время создания
  > Все чекбоксы блокируются во время создания бэкапа
  > При восстановлении: файлы защищённой папки шифруются новым ключом Tink Streaming AEAD нового устройства
