# v1.5 — В разработке
- Экран «О приложении»: иконка, версия, список функций, что нового, логи, GitHub, политика, копирайт
- Экран «Что нового»: раскрывающаяся история версий
- Режим экономии энергии в настройках — отключает все анимации
- Слайдер порога батареи (0–50%) — автоотключение анимаций при низком заряде (по умолчанию 15%)
- ContentObserver: регистрация только на переднем плане (экономия батареи в фоне)
- Свайп назад от левого края на всех экранах: настройки, терминал, библиотека, хранилище, дубликаты, метки, о приложении, функции, темы, анимации
- Метки переработаны: отдельные экраны управления и создания/редактирования с навигацией назад
- Аудит кода: добавлено логирование, удалён мёртвый код, KDoc на 36 классов, документация архитектуры
- Фикс: аудиоплеер играл не тот трек при открытии через имя файла
- Фикс: кнопки аудиоплеера (play/prev/next) не реагировали в полноэкранном режиме
- Фикс: аудио не автоплей при открытии, всегда с начала
- Фикс: прогресс-бар скрыт за кастомным навбаром

# v1.4 — Релиз 2
- 4 оверлейных анимации: Matrix Rain, Снегопад, Звёздное поле, Пылинки с индивидуальными настройками
- Кастомный навбар заменяет системный с настраиваемыми страницами и кнопками
- Радиальное меню: долгий тап для Копировать/Переместить, Удалить/Навсегда, Файл/Папка
- Навигация стрелками с подсветкой курсора, Shift-режим для выделения диапазонов
- Chromecast / DLNA / Miracast с HLS транскодированием (FFmpeg)
- Зеркалирование экрана, слайд-шоу фото и PDF-презентация на ТВ
- Яндекс.Диск / Google Drive / Dropbox: чанковая загрузка с возобновлением
- Стриминг FTP/SFTP/SMB/WebDAV без скачивания
- Полный терминал: VT100/xterm эмуляция, ANSI цвета, SSH через JSch, табы
- Wi-Fi Direct и Bluetooth OPP передача файлов
- FTP-клиент + FTP-сервер, SFTP-клиент, SMB-браузер, WebDAV-браузер
- Защищённая папка: AES-256 шифрование (Android Keystore)
- Стеганография: скрытие файлов в PNG
- Доверенные устройства с автоприёмом
- Редактирование аудио тегов с поиском обложки (Deezer + iTunes)
- Метаданные PDF и FB2 в свойствах файла
- Контрольные суммы: CRC32, MD5, SHA-1, SHA-256, SHA-512 со сравнением хешей
- Архивация «1 в 1»: каждый файл в отдельный ZIP
- Эскизы картинок внутри архивов
- DND в видеоплеере
- Дублирование файлов с 3 режимами назначения
- Пофайловый прогресс для операций с папками с отображением скорости
- Фикс: параллельные загрузки убивали соединения — последовательно через Mutex
- Фикс: троттлинг Яндекса по MIME — загрузка как .tmp + переименование
- Фикс: коррекция offset Dropbox при возобновлении
- Фикс: артефакты AVI в VLC — программный декодер для нестандартных кодеков
- Фикс: переподключение каста после таймаута транскода
- Фикс: HLS version 3 и AAC stereo для совместимости с Chromecast
- Фикс: конфликт свайпа страниц навбара с радиальным жестом

# v1.0 — Релиз 1
- Двухпанельный проводник с drag-and-drop
- Копирование, перемещение, удаление, переименование с разрешением конфликтов
- Корзина с автоочисткой через 30 дней
- Архивы: ZIP, 7z, RAR, TAR, TAR.GZ с защитой паролем
- Текстовый редактор с подсветкой синтаксиса
- PDF-читалка с поиском и навигацией
- Просмотрщик документов: EPUB, FB2, DOCX, ODT, XLSX
- Медиаплеер (VLC): аудио + видео с плейлистом
- Галерея с pinch-to-zoom и свайп-навигацией
- Свойства файлов: EXIF с возможностью удаления
- Полнотекстовый поиск (FTS5) + поиск по содержимому
- Голосовые команды (30+ команд)
- Система жестов (4 свайпа от краёв)
- Анализатор хранилища с круговой диаграммой
- Детектор дубликатов (по хешам)
- Сравнение папок и текстовых файлов (side-by-side diff)
- Менеджер приложений: установленные APK, извлечение/установка
- Закладки, метки, полка (панель избранного)
- Shizuku для доступа к Android/data
- USB OTG хранилище
- Открытие файлов из других приложений (ACTION_VIEW / ACTION_SEND)
