# v0.91 — Энергосбережение, свайп-назад, аудиоплеер
- Режим экономии энергии: отключает анимации, снижает фоновую активность
- Порог батареи: автоотключение анимаций при низком заряде (по умолчанию 15%)
- Свайп назад от левого края на всех экранах настроек, терминала, библиотеки, хранилища, дубликатов, меток
- Аудиоплеер: нет автоплея, всегда с начала, правильный трек при тапе
- Прогресс-бар поднят над кастомным навбаром
- Метки: отдельный экран создания/редактирования, навигация назад
- Аудит кода: добавлено логирование, KDoc комментарии, документация архитектуры

# v0.90 — Анимации-темы
- 4 оверлейных анимации: Matrix Rain, Снегопад, Звёздное поле, Пылинки
- Индивидуальные настройки: скорость, плотность, прозрачность, размер
- Matrix Rain: 5 наборов символов, RGB цвет, эффект свечения
- Опция «Только при зарядке» для каждой анимации
- Автопауза при сворачивании или выключении экрана

# v0.89 — Кастомный навбар, радиальное меню
- Кастомный навбар заменяет системный
- Настраиваемые страницы с 5-7 кнопками
- Радиальное меню: долгий тап для Копировать/Переместить, Удалить/Навсегда, Файл/Папка
- Кастомные иконки для кнопок навбара
- Навигация стрелками с подсветкой курсора
- Shift-режим для выделения диапазонов

# v0.88 — Навигация стрелками, улучшения читалки
- Навигация D-pad стиля стрелками на навбаре
- Умный контраст текста для читаемости на любом фоне
- Улучшения читалки: скроллинг, запоминание позиции

# v0.87 — Облачные загрузки, пакетные операции
- Яндекс.Диск / Google Drive / Dropbox: чанковая загрузка с возобновлением
- Защита от параллельных загрузок (последовательно через Mutex)
- Обход троттлинга Яндекса по MIME-типу (загрузка как .tmp + переименование)
- Сессии загрузки Dropbox с коррекцией offset

# v0.86 — Сетевой стриминг, DND
- Стриминг FTP/SFTP/SMB/WebDAV без скачивания
- HTTP-прокси для FTP (обход бага VLC с кириллицей)
- DND в видеоплеере: глушит уведомления при воспроизведении
- Сохранённые FTP/SMB серверы с учётными данными
