# v1.0 — Файловый менеджер
- Двухпанельный проводник с drag-and-drop
- Копирование, перемещение, удаление, переименование с разрешением конфликтов
- Корзина с автоочисткой через 30 дней
- Архивы: ZIP, 7z, RAR, TAR, TAR.GZ
- Текстовый редактор с подсветкой синтаксиса
- PDF-читалка с поиском и навигацией
- Просмотрщик документов: EPUB, FB2, DOCX, ODT, XLSX
- Медиаплеер (VLC): аудио + видео с плейлистом
- Галерея с pinch-to-zoom и свайп-навигацией
- Свойства файлов: EXIF, аудио теги, метаданные PDF/FB2, контрольные суммы
- Полнотекстовый поиск (FTS5) + поиск по содержимому
- Голосовые команды (30+ команд)
- Система жестов (4 свайпа от краёв)
- Анализатор хранилища с круговой диаграммой
- Детектор дубликатов (по хешам)
- Сравнение папок (side-by-side diff)
- Менеджер приложений: установленные APK, извлечение/установка
- Закладки, метки, полка (панель избранного)
- Shizuku для доступа к Android/data
- USB OTG хранилище
